# allero-backend-work

To run this app we need to prerequisiteis install docker if you don'e have it on your computer, create a .env file with this params

  MONGODB_USER
  
  MONGODB_PASSWORD
  
  MONGODB_DATABASE
  
  MONGODB_LOCAL_PORT
  
  MONGODB_DOCKER_PORT
  
  NODE_LOCAL_PORT
  
  NODE_DOCKER_PORT
  
 
 
`docker-compose up`
  
  
  there u go, your application run on the specified node_local_port, enjoy
